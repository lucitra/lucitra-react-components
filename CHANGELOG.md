## [1.0.0] - 2025-01-07

### Features

- Initial release of @lucitra/react-components
- Navigation components:
  - LanguageSwitcher with multi-language support and RTL handling
  - RegionSwitcher with country/region selection and language integration
- Built-in accessibility features
- Internationalization support
- Theme support (default, dark, high-contrast)
- TypeScript support
- Tree-shakeable build
- Storybook documentation

### Infrastructure

- Automated NPM publishing via GitHub Actions
- Release Please integration for automated versioning
- Comprehensive build pipeline with Rollup
- ESLint and testing setup
